# Álbum Tracker — Copa 2026

PWA para controlar figurinhas da Copa do Mundo 2026 (Panini): completas, faltantes e repetidas para troca, com escaneamento de páginas por IA.

## Publicar no GitHub Pages
1. Crie um repositório novo (ex.: `album-copa-2026`), público.
2. Envie TODOS os arquivos desta pasta para a raiz do repositório
   (index.html, manifest.json, sw.js, os 3 ícones .png, .nojekyll).
3. Settings → Pages → Source: **Deploy from a branch** → Branch: **main** / **/(root)** → Save.
4. Aguarde ~1 min. O link aparece no topo: `https://SEU-USUARIO.github.io/album-copa-2026/`.
5. Abra no Safari → Compartilhar → **Adicionar à Tela de Início**.

## Escanear com IA
No GitHub Pages, abra **Ajustes** e cole sua chave da API Anthropic para ativar a leitura
automática das fotos. A chave fica salva só no seu aparelho. A marcação manual funciona sem chave.
